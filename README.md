# Rasa Nusantara

Demo website statis bertema kuliner khas Indonesia.

## Teknologi
- HTML
- CSS
- JavaScript sederhana

## Fitur
- Landing page
- Card daftar kuliner
- Filter kategori daerah
- Dark mode
- Validasi form sederhana
- Responsive layout

## Pembagian tugas 4 orang
1. Home section dan navbar
2. Daftar kuliner dan card makanan
3. Kategori/filter daerah
4. Tentang, kontak, dan JavaScript
